# 🎓 Course Registration System

A Course Registration System developed using Java, HTML, CSS, Bootstrap, Python Backend, and PostgreSQL.

---

## Features

- 🔐 Login System
- 👨‍🎓 Add Student
- 📋 View Students
- 📚 Add Course
- 📖 View Courses
- 📝 Register Course
- 💾 PostgreSQL Database
- 🎨 Responsive Frontend

---

## Technologies Used

- Java
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Python
- PostgreSQL

## Installation

1. Clone the repository

```bash
git clone https://github.com/Jitenmourya/CourseRegistrationSystem.git
```

2. Open the project

3. Install Python requirements

```bash
pip install flask psycopg2
```

4. Configure PostgreSQL

5. Run

```bash
python python_backend.py
```

6. Open

```
http://localhost:8000
```
